**Project Title**
**Decision-Driven Business Performance Analytics System**

 **Project Description**
This project is an end-to-end business analytics system built using SQL Server, Python, and Power BI. It analyzes e-commerce data to generate actionable insights for business decision-making.

The system processes raw transactional data, performs analysis, and presents results through an interactive dashboard.


 **Objectives**

* Analyze sales and customer data
* Build key performance indicators (KPIs)
* Perform trend and category analysis
* Develop an interactive dashboard
* Generate business insights


 **Tech Stack**

* **SQL Server** – Data storage & transformation
* **Python (Pandas, Matplotlib, Seaborn)** – Data analysis
* **Power BI** – Dashboard visualization

---

**Dataset**

* **Source:** Olist Brazilian E-commerce Dataset (Kaggle)
* ~100,000 orders (2016–2018)
* Multiple tables:

  * Orders
  * Customers
  * Products
  * Payments

---
**Project Workflow**

1. Data Import into SQL Server
2. Data Cleaning & Transformation
3. SQL Analysis & View Creation
4. Python Exploratory Data Analysis
5. Power BI Dashboard Development
6. Insight Generation

---

 **Key KPIs**

* Total Revenue: **14.21M**
* Total Orders: **98.67K**
* Average Order Value: **205.83**
* Total Shipping Cost: **2.36M**
* Total Units Sold: **118K**

---

 **Dashboard Features**

* KPI Cards
* Revenue Trend Analysis
* Top Product Categories
* Customer City Analysis
* Payment Method Distribution
* Year-based Filtering

---

 **Key Insights**

* Revenue shows consistent growth from 2017 to 2018
* Peak sales observed during November (seasonal effect)
* São Paulo is the highest contributing city
* Credit card accounts for ~77% of payments
* Top categories drive majority of revenue

---

**Business Recommendations**

* Focus on high-performing categories
* Optimize logistics costs
* Expand in underperforming regions
* Improve payment infrastructure
* Increase average order value
